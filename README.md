# ancient


##简介

ancient是一款无侵入agent流量治理库，主要基于byte-buddy进行开发，致力解决发布风险问题、内部环境多版本并行问题。
让程序员脱离熬夜发版，提高公司内部研发效率。

##功能介绍

版本灰度

区域灰度

RocketMQ消息队列级别灰度

##支持组件

Spring Cloud Gateway

Spring MVC

Feign

RocketMQ

##使用指南

版本灰度配置

区域灰度配置

RocketMQ消息队列配置

项目运行增加-javaagent:xxx\ancient-agent-0.0.1-SNAPSHOT.jar指令
